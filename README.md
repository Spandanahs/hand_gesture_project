# hand_gesture_project
Hand Gesture Control System

A Python project that allows users to control the mouse using hand gestures captured through a webcam.

Features

- Hand tracking using MediaPipe
- Virtual mouse movement
- Click detection using finger gestures
- Real-time gesture recognition

Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy
- PyAutoGUI

Usage

Run the Python file and allow webcam access. Move your index finger to control the cursor and use a pinch gesture to perform clicks.

Future Enhancements

- Volume control
- Brightness control
- Gesture-based scrolling
-Note: AI tools were used for guidance, debugging, and documentation support during the development of this project.
